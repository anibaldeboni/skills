# Skills

Este repositório contém skills reutilizáveis para uso com agentes no VS Code.

## Como instalar uma skill

Use o comando abaixo na raiz do projeto (ou no diretório onde você quer instalar a skill):

```bash
npx skills add https://github.com/anibaldeboni/skills --skill create-rfc
```

O comando vai abrir o fluxo para selecionar e instalar a skill desejada.

## Verificando a instalação

Após instalar, confirme se a skill foi adicionada na estrutura de pastas do seu projeto.
